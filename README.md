## 🚀 Featured Projects

### 🔍 [Multi-Query Document RAG](https://github.com/kushalvishwak/multi_query_document_rag-)
> Modular Retrieval-Augmented Generation pipeline with multi-query decomposition

A production-ready RAG system that improves retrieval accuracy by generating multiple query variants from a single user question — reducing missed context and improving answer quality.

**Stack:** `Python` `LangChain` `ChromaDB` `Groq` `FAISS`

- ✅ Modular architecture — swap out any component easily
- ✅ Multi-query retrieval for higher recall
- ✅ Fast inference via Groq LPU
